# 步骤三:将文件添加到暂存环境

使用git Add命令将文件添加到暂存环境中。

如果你重新运行git status命令，你将看到git已经将文件添加到暂存环境中(请注意“要提交的更改”行)。

```htlm
mnelson:myproject mnelson$ git status
On branch master

Initial commit

Changes to be committed:
  (use "git rm --cached <file>..." to unstage)

	new file:   mnelson.txt
```
